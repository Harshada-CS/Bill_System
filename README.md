# Billing System (Java Project)

## 📌 Description
This is a Java-based Billing System developed using NetBeans.  
It helps in managing customer billing, generating invoices, and storing data using MySQL database.

## 🚀 Features
- Add Customer Details
- Generate Bills
- Store Data in Database
- Simple User Interface (Java Swing)

## 🛠️ Technologies Used
- Java
- MySQL
- Swing (GUI)
- NetBeans IDE

## ▶️ How to Run
1. Open project in NetBeans
2. Configure MySQL database
3. Run the project

## 📂 Project Structure
- `src/` → Java source files  
- `DBConnection.java` → Database connection  
- `Main UI` → Billing interface
